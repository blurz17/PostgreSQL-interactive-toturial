 # PostgreSQL interactive toturial 🐘 🎮
**PostgreSQL interactive toturial** is an interactive, gamified learning platform designed to teach SQL and database concepts without the boring lectures. Master PostgreSQL through hands-on challenges, leveling up as you write queries, design tables, and connect data!

## 🌟 Features

- **Gamified Learning**: Earn Experience Points (XP), level up your character, and unlock new badges as you complete challenges.
- **Interactive Challenges**: Engage with multiple-choice questions, fill-in-the-blank statements, code snippets, and drag-and-drop puzzles to cement your knowledge.
- **Progressive Curriculum**: Learn everything from the basics of relational databases to complex querying with aggregate functions and foreign keys.
- **Client-Side execution**: Fully runs in the browser using HTML, CSS, and Vanilla JavaScript with no backend dependencies!
- **Modern UI/UX**: Sleek, fully responsive design featuring dark modes and micro-animations for an enjoyable developer experience.

## 🚀 Levels Included

1. **Level 1: Fundamentals** - What is PostgreSQL? Understanding rows, columns, and primary keys.
2. **Level 2: DDL (Data Definition Language)** - Writing your first `CREATE TABLE` and understanding table constraints like `NOT NULL` and `DEFAULT`.
3. **Level 3: Types** - Mastering character, numeric, date/time, and boolean data types.
4. **Level 4: DML (Data Manipulation Language)** - Inserting data with `INSERT INTO` and retrieving it with `SELECT`.
5. **Level 5: Relationships** - Connecting tables logically using Primary and Foreign Keys.
6. **Level 6: SELECT** - Powerful data slicing using `WHERE`, `ORDER BY`, `GROUP BY`, and aggregate functions.

*(More levels can be added easily by expanding the data set in `game.js`!)*

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **CSS3 / Vanilla CSS**: Completely custom styling and animations (no Tailwind or external libraries).
- **Vanilla JavaScript**: State management, level progression, and rendering logic all handled directly via native DOM manipulation.

## 🏁 Getting Started

Since this is a client-side only Web Application, running it is incredibly simple:

1. Clone the repository:
   
git clone https://github.com/blurz17/PostgreSQL-interactive-toturial

2. Open the `index.html` file in your favorite modern web browser.
3. Start learning and earning XP!
